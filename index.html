Tornado Warnings — April 2026

This map displays NWS Tornado Warning alerts from April 2026, across the United States. Each polygon represents a warned area, shape-encoded by severity.

Using real-time NWS alert data as a case study, this tool explores how we can better communicate tornado risk, especially for underserved, rural, or visually impaired users.

About This Page

This map was designed to improve accessibility, clarity, and usability during tornado events.

Features

  • Shape-encoded severity: Each marker shape communicates alert tier
  • Layer toggles: Show/hide warning polygons and centroid markers
  • POI search: Enter addresses to see proximity to warnings
  • Keyboard navigation: Fully accessible interface
